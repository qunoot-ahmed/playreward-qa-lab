# Project Purpose

PlayReward QA Lab is a small portfolio app that demonstrates how to test a rewarded-gaming offer flow.

Version 2 intentionally stays focused:

- One Expo React Native TypeScript app on Expo SDK 54.
- Fictional game: Treasure Quest.
- Multi-area journey: Offers, Offer Details, Game, Completion, Wallet, QA Tools.
- Local-only persistence.
- Manual testing through Expo Go on a physical Android phone.

The goal is to make offer rules, interactive progress, wallet consistency, expiration, and reset behavior easy to inspect and verify without production services or unrelated features.
