# Project Purpose

PlayReward QA Lab is a small portfolio app that demonstrates how to test a rewarded-gaming offer flow.

The project intentionally stays minimal:

- One Expo React Native TypeScript app.
- One screen.
- One offer.
- Local-only persistence.
- Manual testing through Expo Go on a physical Android phone.

The goal is to make the business rules easy to inspect, run, reset, and verify without adding production services or unrelated app features.
