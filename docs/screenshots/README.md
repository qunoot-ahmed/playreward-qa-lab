## Source note

These images were captured from HTML UI previews in `docs/screenshots/` that mirror the Expo Go Version 2 layout and theme (`src/theme.ts` / screen designs). They are reference visuals for GitHub, not raw phone camera dumps.

To replace any image with a real Expo Go device screenshot:

1. Capture the screen on Android.
2. Overwrite the matching filename in this folder.
3. Commit the updated PNG.
